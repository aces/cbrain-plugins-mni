To request additional options, please leave feedback.


Command-specific options:

Output voxel data type specification
  • byte: Write voxel data in 8-bit signed integer format.
  • short: Write voxel data in 16-bit signed integer format.
  • int: Write voxel data in 32-bit signed integer format.
  • float: Write voxel data in 32-bit floating point format.
  • double: Write voxel data in 64-bit floating point format.
  • signed: Write integer voxel data in signed format.
  • unsigned: Write integer voxel data in unsigned format.

Output file format specification
  • dual: Write NIfTI-1 two-file format (.img and .hdr)
  • ASCII: Write NIfTI-1 ASCII header format (.nia)
  • nii: Write NIfTI-1 one-file format (.nii)
  • analyze: Write an Analyze two-file format file (.img and .hdr)
