Civet Options

Not all options listed are provided on CBRAIN
To request additional options, leave feedback.


Pipeline options
  • Template: Define the voxel size in stereotaxic space.
  • Model: Define the model for stereotaxic registration.
  • Restart from this stage: Restart the pipeline from the specified stage. This option appears only when restarting a task.
CIVET options
  • Interpolation: Interpolation method from native to stereotaxic space.
  • N3 distance: N3 spline distance in mm (suggested values: 200 for 1.5T scan; 50 for 3T scan).
  • Head height: Head height in mm for neck cropping. Suggested value: 170. Only available for version 1.1.12 and later.
  • Degrees of freedom for linear registration: Transformation for linear registration.
  • Do NOT build surfaces: Don't build surfaces, if checked.
  • Thickness method: Compute cortical thickness and blur.
  • Kernel size: Default: 20mm for version 1.1.11 and lower and 30mm for version 1.1.12 and later.
  • Resample surfaces: Resample cortical surfaces, if checked.
  • Kernel size areas: Blurring kernel size in mm for resampled surface areas. Default: 40mm. Only available for version 1.1.11 and later.
  • Kernel size volumes: Blurring kernel size in mm for resampled surface volumes. Default: 40mm. Only available for version 1.1.11 and later.
  • Combine surfaces: Combine left and right cortical surfaces, if checked.
VBM options
  • Process VBM files: Process VBM files for analysis, if checked.
  • Blurring kernel size in mm for volume: Blurring kernel size in mm for volume.
  • Run symmetry tool: Run symmetry tools, if checked.
  • Keep cerebellum in VBM maps: Keep cerebellum in VBM maps, if checked.
MINC file input list options
  • Launch CIVET: This checkbox allows you to selectively enable or disable the processing of some files; useful when re-running partial sets of files.
  • Prefix: A simple name or identifier for a set of CIVET outputs. The same name must be used in order to combine several CIVET outputs into a CIVET study.
  • Subject ID: The ID of the subject being processed. These must all be distinct within each prefix.
