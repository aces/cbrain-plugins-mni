Dicom to Minc

Not all options listed are provided on CBRAIN
To request additional options, please leave feedback.


Command-specific options:
  • clobber: Overwrite output files
  • list: Print list of series (don't create files)
  • anon: Exclude subject name from file header
  • cmd: Apply <command> to output files (e.g. gzip)
  • verbose: Print debugging information
  • debug: Print lots of debugging information
  • nosplitecho: Combine all echoes into a single file.
  • splitdynamic: Split dynamic scans into a separate files.
  • opts: Set debugging options - Default value: 0
  • descending: Mosaic sequence is in descending slice order.
  • interleaved: Mosaic sequence is in interleaved slice order.
  • minmax: Honor DICOM pixel minimum and pixel maximum values.
  • stdin: Read file list from standard input.
  • fname: Set format for output file name.
  • dname: Set format for output directory name.
  • usecoordinates: Derive step value from coordinates rather than slice spacing.
