NIfTI to Minc

Not all options listed are provided on CBRAIN
To request additional options, please leave feedback.


Command-specific options:
  • byte: Write voxel data in 8-bit integer format.
  • short: Write voxel data in 16-bit integer format.
  • int: Write voxel data in 32-bit integer format.
  • float: Write voxel data in 32-bit floating point format.
  • double: Write voxel data in 64-bit floating point format.
  • signed: Write integer voxel data in signed format.
  • unsigned: Write integer voxel data in unsigned format.
  • noscanrange: Do not scan data range.
  • quiet: Quiet operation
  • transverse: Assume transverse (ZYX) ordering for spatial dimensions
  • sagittal: Assume sagittal (XZY) ordering for spatial dimensions
  • coronal: Assume coronal (YZX) ordering for spatial dimensions
  • xyz: Assume XYZ ordering for spatial dimensions
  • zxy: Assume ZXY ordering for spatial dimensions
  • yxz: Assume YXZ ordering for spatial dimensions
  • flipx: Invert direction of X (left-right) axis
  • flipy: Invert direction of Y (posterior-anterior) axis
  • flipz: Invert direction of Z (inferior-superior) axis
